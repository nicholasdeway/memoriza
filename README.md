O Memoriza é uma plataforma completa para compra de artigos de papelaria personalizáveis, desenvolvida com foco em produção real. O sistema inclui módulos de autenticação, administração, gerenciamento de pedidos, carrinho, produtos, checkout, cálculo de frete por localidade fixa e integrações com APIs externas (como Mercado Pago, Google e serviços de e-mail), além de APIs RESTful próprias.

Este é um projeto em evolução contínua, com backend em C# / ASP.NET Core e banco de dados PostgreSQL (Supabase).
